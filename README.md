# Toeicvocab
Spring Boot와 React를 활용한 영어 단어 학습 프로그램 프로젝트



## 프로젝트 소개
TOEIC 대비를 준비하는 대학생들이 체계적으로 단어를 암기하고 테스트 할 수 있는 웹 기반 학습 시스템 제공



## 프로젝트 기간

#### 각 팀원들은 맡은 기능을 하이브리드로 진행


- 박태정(PM, Front) 
  - 프로젝트 구상도 설계 및 개발 산출물 관리 및 작성
  - 프론트 구상
  - 관리자 페이지 및 기능구현
  - 메인 페이지 및 기능구현
  - 단어장 페이지 및 기능구현
  - 테스트 페이지 및 기능구현
  - 테스트 결과 페이지 및 기능구현
  - 각 페이지 UI 설계 및 구성
  - 모든 페이지 css 구현
    
- 채예원(발표, Backend)
  - ppt 작성
  - 백엔드 보조
  
- 강민주(Backend)
  - 회의록 작성, 유스케이스 다이어그램 작성 , 백엔드 구상
  - 로그인 및 회원가입 : JWT를 이용한 보안처리 및 백엔드 스프링 시큐리티 사용
  - 게시글 CRUD / 댓글 CRUD 구현
